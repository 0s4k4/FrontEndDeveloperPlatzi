Ejercicio de selectores CSS

Selectores universales

Este selector inicia con * y el selector que se aplica afecta todo el dom

Casi siempre se usa para fijar un backgrouund

Por ejemplo: *{background: whietesmoe}

Selectores de etiqueta

Este selector inicia con el nombre de la etiqueta y se aplica a todas las que esten en el dom

Por ejemplo: section{border:2px solid black;}

Selector por id

Este selector inicia con el id de un objeto en el dom

por ejemplo: #parrafo_seleccionado(color:red)

Selectores de id

Este selector inicia con la clase de un objeto en el dom

Por ejemplo: #parrafo_seleccionado{color:blue}

Selectores de atributos

Este selector inicia con la etiqueta y atributo de un objeto dentro del dom

Por ejemplo input[placeholder="Escribit input"]{border:2px solid orange;}

Selectores combinadores decencientes

Este selector hace que todos los elementos que esten dentros del objeto padre tengan el mismo estilo

Por ejemplo .section_ordenado *{mardgin:20px}

Selectores comibnadores hijo directo

Este selector hace que de todos solo el primer hijo dentro del elemento se modifique

Por ejemplo: #selector_combinador_hijo_directo > article {border:2px solid green;}

Selectores combinadores elemento adyacente

Este selector hace que el elemento siguiente al objeto se modifique

Por ejemplo: #parrago_seleciona2 + p {background-color:aqua;}
